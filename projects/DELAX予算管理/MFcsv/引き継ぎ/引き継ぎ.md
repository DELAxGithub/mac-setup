### ステップ 3: デプロイと権限付与（続き）

1.  **デプロイ設定画面**:
    *   **説明**: `予算管理ダッシュボード v1.0`
    *   **実行するユーザーとして**: `自分 (あなたのメールアドレス)`
    *   **アクセスを許可するユーザー**: **[誰でも]** (※これが重要です。非エンジニアが URL をクリックして使えるようにするため)
2.  **[デプロイ]** ボタンをクリックし、**[ウェブアプリとしてデプロイ]** の URL が表示されます。
3.  この URL をコピーし、ブラウザで開いてみてください。
    *   **初回アクセス時**: 「Google にアクセスを許可しますか？」という警告が出ます。
    *   **[詳細]** > **[このプロジェクトにアクセスを許可する (メールアドレス)]** をクリック。
    *   **権限確認画面**: 下部の **[権限の確認と続行]** > **[有効にする]** を順にクリックします。
4.  これで、CSV アップロード画面が表示されれば完了です。

## 4. 運用マニュアル（非エンジニア向け）

### CSV のアップロード手順
1.  デプロイした URL にアクセスする。
2.  **CSV ファイルをドラッグ＆ドロップ**するか、**「ファイルを選択」**をクリックしてマネーフォワードの仕訳帳 CSV を選ぶ。
3.  「処理中...」と表示され、数秒〜数十秒後に集計結果が表示されます。
4.  画面上部の「集計サマリー」で項目ごとの金額を確認できます。

### 更新ルール（重要）
*   **毎月の予算見直し**: スプレッドシートの `予算設定` シート（または `科目マッピング` の B 列）を編集して、新しい予算目標や分類ルールを設定してください。
*   **CSV の形式変更**: マネーフォワードの CSV ヘッダー名が変わった場合は、`Code.gs` の `parseCSVLine` 関数内のカラムインデックス（`row[0]`, `row[1]` など）を修正する必要があります。

### トラブルシューティング
| エラー内容 | 原因と対策 |
| :--- | :--- |
| **「ファイルが見つかりません」** | CSV ファイルが空、または非 UTF-8 形式です。UTF-8 で保存し直してください。 |
| **「権限がありません」** | デプロイ時のアクセス設定が「誰でも」になっていません。デプロイ設定を再確認してください。 |
| **集計結果がおかしい** | CSV のカラム順序（日付、科目、金額など）が変わっている可能性があります。`Code.gs` の `row[0]`, `row[1]` のインデックスを見直してください。 |
