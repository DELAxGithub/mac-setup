# 仕様書: 経理データ自動取得・反映システム

## 1. プロジェクトの目的
マネーフォワードクラウド会計のAPIから月次の経費データを取得し、Google Sheets APIを使用して予算管理スプレッドシートの「実績」列に自動入力するシステムを構築する。手作業によるCSV集計・転記作業をゼロにすることがゴール。

## 2. 技術スタック
- 言語/環境: Google Apps Script (GAS) または Python
- 外部API: 
  - マネーフォワードクラウド会計 API (OAuth 2.0)
  - Google Sheets API
- 実行トリガー: 手動実行（スプレッドシート上のカスタムメニュー）、または月次バッチ処理

## 3. 機能要件
1. **認証機能:** マネーフォワードAPIのOAuth 2.0認証を行い、アクセストークンを取得・更新・保存する。
2. **データ取得機能:** 指定した対象月（例: 2025年10月）の「勘定科目ごとの合計残高」を取得する。
3. **データマッピング機能:** 取得した一般的な勘定科目（例: 地代家賃、旅費交通費）を、独自の予算管理フォーマットの項目（例: スペイン家賃、金沢事務所費、航空券など）に変換・集計する。
4. **データ書き込み機能:** Googleスプレッドシートの指定したシート・指定した列（対象月の実績列）に、集計した金額を上書きする。

## 4. データ構造・マッピング定義（例）
- `スペイン家賃` <- MFの `地代家賃` (摘要やタグでスペインと判別できるもの)
- `金沢事務所費` <- MFの `地代家賃` (摘要やタグで金沢と判別できるもの)
- `通信費` <- MFの `通信費` 全般

## 5. エラーハンドリングと通知
- APIのレート制限（Rate Limit）に達した場合は適切なリトライ処理を行う。
- 認証エラーやシート書き込みエラーが発生した場合は、ログを出力する（可能であればSlackのWebhookへ通知）。

## 6. AIエージェントへの指示（Prompt）
この仕様書を読み込み、以下の順序で実装を進めてください。
1. プロジェクトのディレクトリ構成案を提示する。
2. マネーフォワードのOAuth認証部分のモジュールを実装する。
3. スプレッドシート読み書き部分のモジュールを実装する。
4. それらを統合するメイン処理を実装する。
