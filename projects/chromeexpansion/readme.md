# Image Viewer - Chrome Extension

Webページ内の画像を自動で順番に表示する Chrome 拡張機能。
ダウンロードせずにブラウザ上でハンドフリーの画像ビューワーとして使える。

## 機能

- ページ内の画像（100x100px以上）を自動検出
- タイマーで次の画像へ自動ジャンプ（1〜15秒間隔）
- 現在位置の表示（3 / 24 枚）
- 前/次ボタンで手動ナビゲーション
- キーボード操作: 矢印キーで前/次、Escで停止
- 最後の画像に到達すると自動停止

## インストール

1. Chrome で `chrome://extensions` を開く
2. 右上の「デベロッパーモード」を有効にする
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. このフォルダを選択

## 使い方

1. 画像が縦に並んだページを開く
2. ツールバーの拡張機能アイコンをクリック
3. スライダーで間隔（秒）を調整
4. 「開始」で自動送り開始、「停止」またはEscで停止
5. ◀ ▶ ボタンや矢印キーで手動移動も可能

## ファイル構成

| ファイル | 内容 |
| --- | --- |
| `manifest.json` | Manifest V3 拡張機能定義 |
| `popup.html` | ポップアップUI（間隔スライダー + ナビボタン） |
| `popup.js` | ポップアップのロジック（content scriptとのメッセージング） |
| `content.js` | 画像検出・スクロール制御スクリプト |

## 技術仕様

- **Manifest Version**: 3
- **権限**: `activeTab`, `storage`
- **対象ページ**: 全URL (`<all_urls>`)
- **画像検出**: `img` タグ、100x100px 以上をフィルタ
- **スクロール**: `scrollIntoView({ behavior: 'smooth' })`
