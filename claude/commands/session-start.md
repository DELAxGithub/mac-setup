セッション開始時のコンテキスト読み込みを行う。

## 手順

1. MEMORY.md を読んで現状を把握
2. 現在のディレクトリの `git status` と `git log --oneline -5` を確認
3. CLAUDE.md があれば読む
4. 前回の残タスクやブロッカーがあれば提示
5. 他エージェントの痕跡チェック:
   - 対象リポジトリに未コミット変更がある場合、警告を出す
   - 直近のコミットに別エージェントの Co-Authored-By がある場合、通知する（Antigravity等）
   - 未コミット変更が自分のものでなさそうなら「別エージェントの作業途中かもしれません」と警告
6. 以下をユーザーに簡潔に報告:
   - 現在のプロジェクト状態（ブランチ、未コミット変更）
   - 他エージェントの痕跡（あれば）
   - 前回からの引き継ぎ事項
   - 今日やるべきこと（あれば）

## ルール

- 日本語で出力
- 長くしない（10行以内目安）
- ユーザーが作業指示を出しやすいように簡潔に
